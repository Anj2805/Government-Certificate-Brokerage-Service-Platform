# Contributing to SevaSetu

We welcome contributions to the SevaSetu project!

## Getting Started
1. Clone the repository.
2. Run `npm install` in the root directory.
3. Run `npm install` in the `frontend/` directory.
4. Copy `.env.example` to `.env` and set up your local MongoDB instance.
5. Familiarize yourself with the backend architecture in `src/` and the frontend architecture in `frontend/src/`.

## Code Guidelines
- Ensure that your code passes all linting rules (`npm run lint`).
- Ensure that all tests pass (`npm run test:all`).
- Please do not submit PRs containing sensitive information or credentials.
- Use meaningful commit messages.

## Bug Reports and Feature Requests
Please use the GitHub Issue tracker to report bugs or submit feature requests. Provide as much detail as possible to help us understand and resolve the issue quickly.
